Honest install

Straightforward, with the friction disclosed.

LidSwitch is not trying to look like an App Store app. It is a small, inspectable utility that asks macOS for admin permission only when it installs, restores, or removes its helper.

  1. 1

    Download DMG

    Get the release DMG from GitHub and open it like any other Mac download.

  2. 2

    Open Anyway

    If Gatekeeper blocks the first launch, approve it in macOS Security settings.

  3. 3

    Approve helper

    macOS asks for admin permission before LidSwitch installs the local helper.

  4. 4

    Restore anytime

    Turn it off, restore sleep defaults, or uninstall from the menu bar.

For technical friends

Inspect it. Install it. Remove it.

The point is not mystery. The point is a tiny switch with a clear power policy: plugged-in work can keep running, battery sleep remains normal unless you explicitly choose otherwise.

No credentials stored

LidSwitch does not collect passwords, tokens, accounts, or personal data.

Open to inspect

The app, helper, scripts, and docs live in the GitHub repository.

Battery opt-in

Battery lid-close sleep stays allowed unless you deliberately enable it.

Easy to remove

Restore and Uninstall are first-class controls in the menu bar panel.

Free to use. Source available.

If it helps your Mac finish the job, use it.

Requires an Apple Silicon Mac with macOS 14 or newer. Built for people comfortable reviewing source, approving a local helper, and removing the tool if they change their mind.
